How it works & notes

To / CC: The link sets To = Recipient + LandscapeDivisionLeadership@hunterindustries.com and CC = Recipient Manager + Submitted. Names use display names, resolved against your company address book (GAL) when Outlook opens. If a name doesn't auto-resolve, Outlook underlines it so you can pick the right person.

Signature: Whatever you type in the Signature box is appended (as plain text) to the end of every email. Rich/graphical signatures (logos, colors) can't go through a mailto: link — use plain text only.

Length limit: Excel hyperlinks and Windows mailto: are capped near ~2,000 characters. The AI is told to keep emails short so nothing is cut off. Rows that still exceed the limit are flagged trimmed.

Privacy & key: The Gemini API key lives only on the server (Vercel env var) — it is never shown in the browser. Only the recipient name + comments + your prompt are sent to Google's Gemini API to write each message.
